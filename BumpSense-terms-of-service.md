### 2.2. Назначение Приложения

Приложение предназначено для:

- записи треков движения с использованием GPS-приёмника устройства;
- определения неровностей дорожного покрытия с помощью акселерометра;
- визуализации собранных данных на карте;
- управления сохранёнными треками.

Все права, прямо не предоставленные настоящим Соглашением, остаются за Разработчиком.

### 4.2. Сторонние компоненты

Приложение использует следующие сторонние библиотеки и компоненты:

- **MapLibre** — для отображения карт (лицензия BSD);
- **Jetpack Compose** — для пользовательского интерфейса (лицензия Apache 2.0);
- **Room Database** — для локального хранения данных (лицензия Apache 2.0);
- **Kotlin Coroutines** — для асинхронных операций (лицензия Apache 2.0).

Все сторонние компоненты используются в соответствии с условиями их лицензий. Тексты лицензий доступны по ссылкам на официальные сайты соответствующих проектов.
