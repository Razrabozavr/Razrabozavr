# Политика конфиденциальности BumpSense

**Последнее обновление:** 1 июля 2026 г.  
**Версия:** 1.0

---

## 1. Общие положения

Настоящая Политика конфиденциальности («Политика») описывает, какие данные собирает, обрабатывает и хранит мобильное приложение **BumpSense** («Приложение»), разработанное [ВАШЕ ИМЯ] («Разработчик»), а также как пользователь может управлять своими данными.

Используя Приложение, вы соглашаетесь с условиями настоящей Политики. Если вы не согласны с какими-либо положениями, пожалуйста, не используйте Приложение.

---

## 2. Какие данные собираются

### 2.1. Данные о местоположении

Приложение запрашивает следующие разрешения для работы с геолокацией:

- `ACCESS_FINE_LOCATION` — точное местоположение (GPS)
- `ACCESS_COARSE_LOCATION` — приблизительное местоположение (сеть)
- `ACCESS_BACKGROUND_LOCATION` — доступ к местоположению в фоновом режиме (необходимо для длительной записи треков)

**Цель сбора:** запись GPS-треков, отображение текущей позиции на карте, привязка данных акселерометра к координатам.

### 2.2. Данные с датчиков устройства

Приложение использует встроенный **акселерометр** для:
- определения вибраций и неровностей дорожного покрытия;
- вычисления индекса неровности (BumpIndex) в диапазоне 0–100.

Данные акселерометра **не покидают устройство** и не передаются третьим лицам.

### 2.3. Данные, создаваемые пользователем

В процессе использования Приложение создаёт:
- GPS-треки (координаты, время, скорость, индекс неровности);
- настройки пользователя (тема, интервал GPS, пороги чувствительности).

### 2.4. Технические данные

Приложение использует интернет **только** для:
- загрузки картографических тайлов с серверов OpenStreetMap через библиотеку MapLibre;
- проверки состояния сети.

**Приложение НЕ собирает и НЕ передаёт:**
- идентификаторы устройства (IMEI, Android ID, Advertising ID);
- данные об использовании (аналитику);
- персональные данные (имя, email, телефон);
- рекламные идентификаторы.

---

## 3. Как используются данные

Все собранные данные используются **исключительно** для:

- отображения треков на карте в реальном времени;
- сохранения истории поездок в локальной базе данных;
- анализа качества дорожного покрытия;
- предоставления пользователю возможности экспорта данных в формате GeoJSON.

Данные **НЕ используются** для:
- отслеживания местоположения пользователя в коммерческих целях;
- передачи третьим лицам;
- таргетированной рекламы;
- профилирования поведения.

---

## 4. Хранение и защита данных

### 4.1. Локальное хранение

Все данные хранятся **исключительно на устройстве пользователя** в защищённой базе данных SQLite (Room). Данные **не отправляются** на серверы Разработчика.

### 4.2. Резервное копирование

База данных включена в стандартный механизм резервного копирования Android:
- **Auto Backup** (Google Drive) — для устройств с API < 31;
- **Cloud Backup** и **Device Transfer** — для устройств с API 31+.

Пользователь может отключить резервное копирование в настройках Android:
**Настройки → Система → Резервное копирование**.

### 4.3. Экспорт данных

Пользователь может экспортировать свои треки в формате **GeoJSON** через встроенную функцию «Экспорт». Экспортированные файлы контролируются пользователем и могут быть удалены в любой момент.

### 4.4. Удаление данных

Пользователь может:
- удалить отдельные треки через режим редактирования;
- полностью очистить базу данных через меню приложения;
- удалить Приложение вместе со всеми данными через настройки устройства.

## 5. Сторонние сервисы

### 5.1. OpenStreetMap и MapLibre

Приложение использует бесплатные картографические сервисы:
- **MapLibre** — библиотека для отображения карт (https://maplibre.org);
- **OpenStreetMap** — источник картографических данных (https://www.openstreetmap.org).

При загрузке карт могут передаваться только координаты видимой области карты. Подробнее: [Privacy Policy OSMF](https://wiki.osmfoundation.org/wiki/Privacy_Policy).

### 5.2. Google Play Services

Для получения геолокации используется **Fused Location Provider** от Google. Обработка данных регулируется политикой конфиденциальности Google: https://policies.google.com/privacy.

## 6. Данные детей

Приложение не предназначено для лиц младше **13 лет**. Разработчик не собирает сознательно персональные данные детей.

## 7. Изменения в Политике

Разработчик может периодически обновлять настоящую Политику. О существенных изменениях будет сообщено через:
- обновление приложения в Google Play;
- изменение даты «Последнее обновление» в начале документа.

Продолжение использования Приложения после внесения изменений означает согласие с новой редакцией Политики.
