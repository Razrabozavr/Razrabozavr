# 🚗 BumpSense

**Анализатор неровности дорог для Android**

BumpSense — это Android-приложение для записи GPS-треков с автоматической фиксацией неровностей через встроенный акселерометр.

## ✨ Основные возможности

- 📍 **Запись GPS-треков** в реальном времени с настраиваемым интервалом
- 📊 **Анализ неровностей** через акселерометр с вычислением индекса BumpIndex (0–100)
- 🎨 **Цветовая индикация** на карте: ровная дорога (🟢), небольшие неровности (🟡), значительные неровности (🔴)
- 🗺️ **Бесплатные карты OpenStreetMap** через MapLibre без API-ключей
- 📤 **Экспорт/импорт** треков в формате GeoJSON
- ⚙️ **Гибкие настройки**: интервал GPS, порог чувствительности, автосохранение
- 🌓 **Тёмная и светлая темы**

## 🔧 Технические детали

- **Платформа:** Android 8.0+ (API 26+)
- **Технологии:** Kotlin, Jetpack Compose, Material 3, MapLibre, Room, Kotlin Coroutines + Flow
- **Архитектура:** MVVM + Clean Architecture
- **Оптимизация:** Batch-операции с БД, упрощение геометрии (RDP), дебаунс обновлений карты
