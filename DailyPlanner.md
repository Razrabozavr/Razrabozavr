# DailyPlanner

**DailyPlanner** — это современное Android-приложение для управления личными задачами (To-Do List), созданное с упором на приватность, удобство и производительность.

Главная философия приложения: ваши данные принадлежат только вам. DailyPlanner не использует облачные серверы, не собирает аналитику и не требует регистрации. Все задачи хранятся локально в зашифрованной базе данных SQLite на вашем устройстве.

---

## ✨ Ключевые особенности

- **🔒 Privacy-First**  
  Никаких трекеров, рекламы и сбора персональных данных. Полностью оффлайн-работа.

- **Управление жестами**  
  Свайпы для быстрой смены статуса, долгое нажатие для удаления и двойной тап для завершения задачи.

- **🎨 Material 3 UI**  
  Современный адаптивный дизайн с поддержкой светлой, тёмной и системной тем, а также настройкой масштаба интерфейса.

- **Надёжное хранение**  
  Встроенный экспорт и импорт базы данных, а также автоматическое создание локальных резервных копий.

- **🔄 RuStore SDK**  
  Нативная проверка и установка обновлений через магазин приложений RuStore.

---

## 🛠 Технологический стек

| Компонент            | Технологии                                      |
|----------------------|-------------------------------------------------|
| **Язык**             | 100% Kotlin                                     |
| **UI**               | Jetpack Compose + Material 3                    |
| **Архитектура**      | MVVM + Clean Architecture                      |
| **База данных**      | Room (SQLite) + Kotlin Coroutines & Flow       |
| **Обновления**       | RuStore In-app Updates SDK                     |
