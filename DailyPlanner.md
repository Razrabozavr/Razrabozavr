# 📱 ЕЖЕДНЕВНИК

**Простое приложение для управления задачами**

---

## Основные возможности

### 📝 Создание задач
Создавайте задачи с кратким и полным описанием.

---

### ⏰ Гибкая система сроков
Предустановленные сроки выполнения:

- ♾️ **Бессрочные задачи**
- 🔢 **На 3 дня**
- 📅 **На неделю**
- 📆 **На месяц**
- 🎯 **Произвольная дата**

---

### 📊 Статусы задач

| Статус | Описание | Оформление |
|--------|----------|------------|
| 🕐 **В Работе** | Активные задачи | Сине-бирюзовый градиент |
| ⏸️ **Отложено** | Задачи на паузе | Оранжево-жёлтый градиент |
| ❌ **Отменено** | Отменённые задачи | Красно-розовый градиент |
| ✅ **Завершено** | Выполненные задачи | Зелёно-бирюзовый градиент |

---

### 👆 Быстрые действия

- **Swipe** → Перемещение в отложенные / в работу
- **Double Tap** → Быстрое завершение задачи
- **Long Press** → Быстрое удаление задачи

---

### 🎨 Визуальная подсветка

Автоматическая подсветка задач:

- 🔴 **Просроченные задачи**
- 🟢 **Сегодняшние задачи**

---

### 🔒 Безопасность данных

Все данные хранятся **локально** на вашем устройстве.  
Никаких серверов, облаков и передачи данных третьим лицам.
